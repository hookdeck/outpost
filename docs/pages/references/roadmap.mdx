## ALPHA

### Core Functionality

- ✅ Tenant & Destination APIs (Redis storage)
- ✅ Destination registry pattern
- ✅ Event ingestion: Publish API & PublishMQ
- ✅ Event delivery
- ✅ Log (events & attempts) persistence logic
- ✅ PostgreSQL log storage
- ✅ User Portal v0.1

See Outpost [Concepts](/docs/concepts) and [Features](/docs/features) for more information.

### Internal Message Queues

- ✅ AWS SQS
- ✅ RabbitMQ

### Publish Message Queues

- ✅ AWS SQS
- ✅ RabbitMQ

### Destination Types

- ✅ Webhooks
- ✅ AWS SQS
- ✅ RabbitMQ
