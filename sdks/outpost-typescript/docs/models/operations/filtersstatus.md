# FiltersStatus

Filter by attempt status(es). Use bracket notation for multiple values (e.g., `filters[status][0]=success&filters[status][1]=failed`).


## Supported Types

### `operations.FiltersStatusEnum1`

```typescript
const value: operations.FiltersStatusEnum1 = "success";
```

### `operations.FiltersStatusEnum2[]`

```typescript
const value: operations.FiltersStatusEnum2[] = [
  "failed",
];
```
