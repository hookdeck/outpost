# TLS

Whether to use TLS connection (amqps). Defaults to "false".

## Example Usage

```go
import (
	"github.com/hookdeck/outpost/sdks/outpost-go/models/components"
)

value := components.TLSTrue
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `TLSTrue`  | true       |
| `TLSFalse` | false      |
